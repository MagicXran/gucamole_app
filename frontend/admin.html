Worker 管理

Worker 配置向导
  • 节点组 是“同一类运行环境”的抽象，比如同一个软件、同一套许可证、同一类机器。
  • Worker 节点 是一台具体 Windows 主机，必须填对主机名、共享工作区和 scratch 路径。
  • 脚本应用绑定 在“应用管理”里完成：启用脚本模式后，把软件预设、执行器和 Worker 组绑在一起。

脚本模式配置原则
  • 普通 RemoteApp 只管 RDP 参数；启用脚本模式 后才需要再选软件预设、执行器和 Worker 组。
  • 软件预设解决“跑什么环境”，Worker 组解决“去哪台机器跑”，执行器解决“怎么拉起脚本”。
  • 如果客户看不懂某个字段，说明这个字段该加解释，而不是让客户猜。
